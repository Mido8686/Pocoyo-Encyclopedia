Welcome to **Pocoyo Encyclopedia**!

This is a fan-made, Pocoyo-related encyclopedia written entirely in Markdown (`.md`) format.

---

## Status
**This wiki is currently experimental.**  
Content, layout, and structure are subject to change while development continues.

---

## About
Pocoyo Encyclopedia aims to collect and document everything about the *Pocoyo* universe —  
including characters, episodes, behind-the-scenes information, and fan observations.

---

## Structure
All pages are stored as `.md` files inside this repository.  
You can browse them directly through folders such as:

- `characters/`

---
